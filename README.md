# zhandayeva.com

Static HTML/CSS, no build step. Hosted on GitHub Pages with a custom domain.

## Pages
- `index.html` — about
- `publications.html` — peer-reviewed articles, book chapters, other publications
- `work-in-progress.html` — working papers
- `teaching.html` — courses and teaching philosophy
- `cv.html` — CV / resume viewer
- `contact.html` — contact info

## Local preview
Open `index.html` in any browser, or run a tiny local server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.
